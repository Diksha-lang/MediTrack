MediTrack - Clinic and Appointment Management System

This project is a simple Java program to manage patients, doctors, appointments, and billing.

Features:
- Add and view patients
- Add doctors
- Book appointments
- Generate bill
- Save and load data

Project Structure:
- entity
- service
- util
- interfaces
- exception
- test

How to run:
1. Go to src folder
2. Compile: javac com/airtribe/meditrack/Main.java
3. Run: java com.airtribe.meditrack.Main
